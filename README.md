## How to Run
1. Clone or download the project files.
2. Open the project in **Unity Hub** ‚Üí Add project from disk.
3. In Unity, open the `MainScene` (or your active scene).
4. Press **Play** in the Editor to test.  
   OR  
   To build:
   - Go to **File ‚Üí Build Settings ‚Üí Select Andriod ‚Üí Make Sure it is set to both**.
   - Click **Build and Run**.
